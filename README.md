# FinTrack - Personal Finance Expense Tracker

FinTrack is a premium, high-performance Personal Finance Expense Tracker built as a Capstone Project. It leverages modern web technologies to provide a seamless and visually stunning experience for managing personal finances.

## 🚀 Features

- **Dynamic Dashboard**: Real-time overview of balance, income, and expenses with interactive charts.
- **Advanced Transactions**: Search, filter, and sort transactions with a clean, responsive table.
- **Redux State Management**: Robust state handling with persistence using Redux Toolkit.
- **Premium UI/UX**:
  - Dark Mode support with system preference detection.
  - Glassmorphic design elements.
  - Smooth animations using Framer Motion.
  - Fully responsive layout (Mobile & Desktop).
- **Data Visualization**: Insightful trends and category breakdowns using Chart.js.

## 🛠️ Tech Stack

- **Frontend**: React (Vite)
- **State Management**: Redux Toolkit
- **Routing**: React Router DOM
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Charts**: Chart.js & React-Chartjs-2

## 📦 Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 📂 Project Structure

```
src/
├── components/     # Reusable UI components
│   ├── dashboard/  # Dashboard specific components
│   ├── layout/     # Sidebar, Navbar, etc.
│   └── transactions/ # Transaction forms and modals
├── store/          # Redux store and slices
├── pages/          # Main application pages
├── hooks/          # Custom React hooks
└── utils/          # Helper functions
```
